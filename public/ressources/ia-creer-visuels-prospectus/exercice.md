# Exercice pratique — Créer un visuel avec l’IA

## Objectif

Créer un visuel professionnel pour votre entreprise à l’aide d’un outil d’IA.

## Consigne

1. Choisissez un outil (Canva IA, Microsoft Designer ou autre).
2. Choisissez un format : post Instagram, flyer A5 ou bannière de site web.
3. Rédigez un prompt détaillé avec : le format, le style, les couleurs, l’ambiance et ce que vous ne voulez pas.
4. Générez au moins 3 variantes.
5. Sélectionnez la meilleure et notez les éléments à corriger ou ajouter manuellement (texte, logo, informations de contact).

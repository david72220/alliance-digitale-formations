# Exercice — Créer 3 modèles de rédaction IA

## Consigne

Choisissez 3 types de documents récurrents dans votre activité (ex : relance client, CR réunion, fiche procédure).

Pour chacun :
1. Rédigez un prompt standard.
2. Générez un exemple avec l’IA.
3. Notez ce qu’il faut vérifier ou corriger avant envoi.

## Livrable

Une petite bibliothèque de 3 prompts testés.

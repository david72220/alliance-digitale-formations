# Exercice — Sécuriser ses emails

## Consigne

Dans votre PME, identifiez :
1. Les 3 types de documents les plus sensifs envoyés par email.
2. Les risques associés à chacun.
3. Une alternative de partage plus sécurisée.

## Livrable

Un tableau avec ces 3 cas : document, risque, solution proposée.

## Points de vigilance

- Le principe de base est : **aucun accès par défaut**.
- Les accès doivent être révisés à chaque changement de poste.
- Les comptes de personnes parties doivent être désactivés rapidement.
- Gardez une trace des révisions pour les contrôles éventuels.

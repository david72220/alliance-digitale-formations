# Exercice pratique — Identifier et cartographier ses tâches répétitives

## Objectif
Repérer les 3 tâches les plus chronophages de votre poste et évaluer leur potentiel d'automatisation.

## Consigne

1. Listez toutes les tâches que vous réalisez au moins une fois par semaine et qui prennent plus de 15 minutes.
2. Pour chaque tâche, complétez le tableau suivant :

| Tâche | Fréquence | Temps/semaine | Répétitive | Données identifiables | Score automatisation |
|---|---|---|---|---|---|
| Exemple : Saisie des factures | Quotidien | 4 h | Oui | Email + Drive | Facile |
| **Tâche 1** | | | | | |
| **Tâche 2** | | | | | |
| **Tâche 3** | | | | | |
| **Tâche 4** | | | | | |
| **Tâche 5** | | | | | |

3. Choisissez **la tâche prioritaire** en justifiant votre choix :
   - Temps gagné potentiel
   - Simplicité technique
   - Faible risque en cas d'erreur

4. Rédigez une fiche processus pour cette tâche :
   - **Déclencheur** : quand la tâche démarre-t-elle ?
   - **Étapes actuelles** : décrivez chaque action.
   - **Données utilisées** : quels outils, quels fichiers ?
   - **Cas d'exception** : que se passe-t-il si une information manque ?
   - **Résultat attendu** : à quoi reconnaît-on que la tâche est bien faite ?

## Livrable attendu
Un document avec le tableau, le choix prioritaire et la fiche processus.
